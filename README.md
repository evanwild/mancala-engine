# Mancala Engine

What is the longest possible game of [Mancala](https://www.youtube.com/watch?v=OX7rj93m6o8)?

The best I've found is [503 moves long](https://github.com/evanwild/mancala-engine/blob/main/gamelog.txt).

## Usage

```sh
git clone https://github.com/evanwild/mancala-engine.git
cd mancala-engine
./run.sh
```
